# Frontend Structure

This directory contains the frontend code for the Safety SaaS application.

## Structure
- `pages/`: Main HTML pages
- `components/`: Reusable UI components
- `css/`: Stylesheets
- `js/`: JavaScript files
- `ai_modules/`: AI feature UI modules
- `ai_tools/`: Advanced AI tools
- `assets/`: Images, icons, fonts

Refer to each folder for more details.
